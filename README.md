# URL Shortener

A simple web application that shortens URLs using a **Flask** backend and a basic **HTML/JavaScript** frontend. Users can input a long URL and receive a shortened version. If the URL has already been shortened, the application returns the existing shortened URL.

## Features
- Shorten any given URL.
- Returns the same shortened URL if the input URL was previously shortened.
- Provides a copy button to easily copy the shortened URL.
- Basic frontend with input validation.

## Technologies Used
- **Backend**: Flask (Python)
- **Frontend**: HTML, JavaScript, CSS
- **Database**: Python dictionary (In-memory storage for simplicity)
